1. С помощью текстового редактора создать файл, содержащий текст. Длина строки текста не должна превышать 80 символов. Это исходный файл.
2. Программа должна:
1) информацию исходного файла записать в выходной файл;
2) реализовать действия, указанные в индивидуальном соответствующую информацию в выходной файл.
3. Имя исходного  файла задает пользователь во время выполнения или в командной строке.
4. Имя выходного файла: первые символы - не более трех символов из имени исходного файла, а остальные символы _out.
5. Выходной файл должен иметь расширение .dat.
6. Программа должна обеспечивать по запросу пользователя:
1) вывод текста исходного файла на экран дисплея;
2) вывод текста выходного файла на экран дисплея. 
7. Законченные последовательности действий оформить в виде функций. Все необходимые данные для функций должны передаваться им в качестве параметров. Использование глобальных переменных в функциях не допускается.
8. Определения функций и главную функцию разместить в двух отдельных файлах.
